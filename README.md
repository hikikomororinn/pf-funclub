pf-funclub
====
ファンクラブを作成できるプラットフォーム型サイトです。
- クリエイターが自らのファンクラブを作成
- 希望者がファンクラブを購読
- 購読することで、該当クリエイターのファンクラブページを閲覧可能
という機能を持っています。
## 導入したプラグイン・API・FWなど
- Firebase Cloud Firestore
- Firebase Functions
- Firebase Authorization
- Next.js(JavaScript FW)
- Material-UI
- Stripe API
